В этом задании вы познакомитесь с артефактами,
которые создаются в процессе сборки приложения
(при выполнении команды Gradle build).

После выполнения команды Gradle build в вашем модуле должна
появиться папка build. В среде разработки Idea она подсвечивается
оранжевым цветом.

В этой папке расположены артефакты созданные Gradle в процессе сборки приложения.

Откройте папку build. В ней можно найти следующие папки:
- classes - тут лежит bytecode откомпилированных классов
- classes/java/main - bytecode продакшн классов
- classes/java/test - bytecode тестовых классов
- generated - тут будет лежать сгенерированный при компиляции код (о нем вы узнаете чуть позже)
- libs - тут лежит собранный из продакшн кода jar файл
- reports - содержит сгенерированный отчет по тестам
- resources - содержит файлы ресурсов используемые приложением (о них вы узнаете чуть позже)
- test-results - содержит результаты выполнения тестов.

Ваше задание: исследуйте содержимое папки build.

PS: не забудьте по окончанию выполнения шага создать и заполнить
файл /documentation/Task_x_description.txt. Формат заполнения этого
файла описан в task_1 в файле Learning_reflection.txt.
