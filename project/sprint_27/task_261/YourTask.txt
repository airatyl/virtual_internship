В предыдущих шагах мы завершили разработку BlackList микро сервиса.

Ваша задача: добавить в Insurance Calculator проверку внесена
ли персона в "черный список" или нет. Для этого нужно вызвать
BlackList REST API для каждой персоны в запросе. И если хоть
одна персона окажется в "черном списке" то для нее вернуть
ошибку (для этого добавьте новый код ошибки).

Учтите тот момент, что BlackList REST API вызов это синхронная
интеграция. Будьте готовы к тому, что микро сервис BlackList
может просто не работать так как это другое приложение и у нас
нет никаких гарантий, что оно будет всегда работать. В случае
если BlackList микро сервис не доступен Insurance Calculator
должен выдавать общую ошибку, которая говорит о том что по техническим
причинам расчет страховой премии в данный момент не доступен.
Для этого заведите отдельный код ошибки!
