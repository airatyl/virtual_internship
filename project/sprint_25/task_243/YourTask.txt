Создание заглушки для отправки сообщений в RabbitMQ.

В пакете core создайте под пакет messagebroker.

В пакете messagebroker создайте интерфейс

public interface ProposalGeneratorQueueSender {
    void send(AgreementDTO agreement);
}

У нас будет две реализации этого интерфейса.
В этом шаге мы создадим реализацию этого интерфейса заглушку,
а в следующих шагах создадим реальную реализацию. Заглушка
будет использоваться для сред, где нет реального брокера сообщений.

Создайте реализацию этого интерфейса, которая просто
превращает AgreementDTO в JSON и печатает в лог (заглушка).

Воспользуйтесь аннотацией @Profile для указания, что эта
заглушка используется на всех Spring Profiles. Можете просто
их перечислить. В дальнейшем мы это изменим.

Вызовите эту заглушку из TravelCalculatePremiumServiceImpl,
конечно для получения зависимости воспользуйтесь интерфейсом.

Протестируйте работу заглушки.
